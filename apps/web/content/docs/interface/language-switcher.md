# Language Switcher

Switch between languages to change your learning context.

## How It Works

The language switcher in the top bar lets you change your L1 (native language) and L2 (target language). This affects all content, subtitles, and dictionary results.

## Features

- **L1 Selector** — Choose your native/UI language
- **L2 Selector** — Choose the language you're learning
- **Swap Button** — Quickly swap L1 and L2
- **60+ Languages** — Extensive language support

## Script Support

Some languages support multiple writing systems:

| Language | Scripts |
|---|---|
| Chinese | Simplified, Traditional |
| Japanese | Kanji, Kana, Romaji |
| Korean | Hangul, Hanja |
| Vietnamese | Quốc Ngữ, Chữ Hán |

## Tips

- Your language pair is saved between sessions
- Switching L2 changes all content recommendations
- Use the swap button to practice both directions
