# Video Player

Language Player's video player is designed specifically for language learning, with features that go far beyond a standard video player.

## Supported Content Sources

- **YouTube** — 600,000+ videos available
- **TV Shows** — Browse by show, season, and episode
- **Live TV** — Watch live broadcasts in your target language
- **Music** — Music videos with synced lyrics

## Interactive Subtitles

### Dual Subtitles (L1 + L2)
Every video displays subtitles in both your native language (L1) and target language (L2). This lets you understand the content while absorbing the target language.

### Click-to-Seek
Click any subtitle line to jump directly to that moment in the video.

### Auto-Pause
The player can automatically pause after each subtitle line, giving you time to process new vocabulary.

## Playback Controls

| Control | Description |
|---|---|
| Play/Pause | Standard playback control |
| Auto-Pause | Pause after each subtitle line |
| Skip Back/Forward | Jump between subtitle segments |
| Playback Speed | Slow down (0.5×) or speed up (2×) |

## Tokenization

Subtitles are **tokenized** — each word becomes clickable. Click any word to:
- See its dictionary definition
- Hear its pronunciation
- Save it to your vocabulary list
- See example sentences

## Keyboard Shortcuts

| Key | Action |
|---|---|
| `→` | Next subtitle line |
| `←` | Previous subtitle line |
| `Space` | Play / Pause |
