# Getting Started with Language Player

Language Player helps you learn languages through authentic video content with interactive dual subtitles.

## What is Language Player?

Language Player is a language learning platform that uses real videos — from YouTube, TV shows, movies, and more — as the core learning material. Watch content you enjoy while building vocabulary, improving listening comprehension, and mastering grammar naturally.

## Supported Languages

Language Player supports **60+ languages** across **207+ language pairs**. You can learn any language from any native language.

### Popular Language Pairs

- English → Spanish, French, Japanese, Chinese, Korean
- Chinese → English, Japanese, Korean
- Japanese → English, Chinese
- Korean → English, Chinese

## Key Features

- **Interactive Dual Subtitles** — See both your native language (L1) and target language (L2) subtitles side by side
- **Popup Dictionary** — Click any word to see definitions, pronunciation, examples, and conjugations
- **Saved Words** — Save vocabulary for later review with spaced repetition
- **Smart Difficulty Tracking** — Words are tracked by CEFR, HSK, or JLPT levels
- **Multiple Content Types** — Videos, music, live TV, EPUB books, and web articles
