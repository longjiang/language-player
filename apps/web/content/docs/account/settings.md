# Settings

Configure your Language Player experience.

## Display Settings

- **Theme** — Light, dark, or system default
- **Font** — Default, serif, or sans-serif
- **Text Size** — Adjust subtitle and transcript text size
- **Translation** — Show/hide L1 translation below subtitles

## Playback Settings

- **Auto-Pause** — Pause after each subtitle line
- **Playback Speed** — Adjust video speed (0.5× to 2×)
- **Subtitle Delay** — Offset subtitle timing

## Speech Settings

- **Pronunciation Voice** — Choose voice for word pronunciation
- **Auto-Play Pronunciation** — Play pronunciation on word click

## Phonetics Settings

- **Show Phonetics** — Display pronunciation guides above words
- **Phonetics Mode** — Show above word, replace word, or hide
- **Show For** — All words or only difficult words

## Review Settings

- **Cards Per Day** — Number of new review cards daily
- **Quiz Mode** — Hide saved words as blanks to test memory
