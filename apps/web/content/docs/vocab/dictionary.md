# Dictionary

The built-in dictionary is one of Language Player's most powerful features. It provides instant lookups for any word in your target language.

## How to Use the Dictionary

1. **Click any word** in the subtitles or transcript to open the popup dictionary
2. **Use the search bar** at `/dictionary` to look up any word manually
3. **Browse recent searches** to quickly revisit words you've looked up

## Dictionary Information

For each word, the dictionary shows:

### Core Information
- **Definitions** — Clear, concise meanings in your native language
- **Pronunciation** — Audio playback and phonetic notation (Pinyin, Romaji, IPA, etc.)
- **Part of Speech** — Noun, verb, adjective, etc.

### Advanced Information
- **Example Sentences** — See the word used in real contexts
- **Collocations** — Common word combinations
- **Verb Conjugations** — Full conjugation tables for verbs
- **Character Decomposition** — For Chinese characters, see radicals and components

## Script Support

Language Player supports multiple writing systems:

| Language | Scripts |
|---|---|
| Chinese | Simplified, Traditional, Pinyin |
| Japanese | Kanji, Hiragana, Katakana, Romaji |
| Korean | Hangul, Hanja, Romanization |
| Vietnamese | Quốc Ngữ, Chữ Hán |

You can configure script preferences in [Settings](/settings).

## Word Difficulty Levels

Words are classified by proficiency level:

- **CEFR** — A1 through C2 (European languages)
- **HSK** — Levels 1–6 (Chinese)
- **JLPT** — N5 through N1 (Japanese)
