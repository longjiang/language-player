# Web Reader

Read any webpage with interactive word lookup.

## How It Works

Enter a URL and the Web Reader extracts the article text, making every word clickable with dictionary lookup. This lets you read real-world content in your target language.

## Features

- **URL Input** — Paste any article URL
- **Clean Reading View** — Extracted text without distractions
- **Dictionary Integration** — Click any word for definitions
- **Save Vocabulary** — Add words to your list while reading

## Tips

- Use news sites in your target language for current, relevant content
- Wikipedia articles are great for building topic-specific vocabulary
- Save the page to read offline later
